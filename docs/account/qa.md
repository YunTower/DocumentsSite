# 云塔账号通行证 常见问题Q&A

## 1. 云塔会保存第三方账号的敏感信息吗
云塔承诺不会去保存和使用您在第三方平台的敏感信息（包括账号，密码以及Token等），**仅仅是获取第三方平台的uid**用来与云塔第三方账号关系数据库进行比对，判断你有没有绑定第三方平台到云塔账号或此第三方账号是否绑定了云塔账号，因此不用担心这个问题。
如果您担心云塔会保存和使用，您大可以选择默认的云塔账号登录或不使用云塔。
由于编程猫官方没有开放的API接口，因此编程猫账号登录都是通过解析编程猫登录的逻辑实现的，因此可能会有不稳定的情况。
编程猫登录与后端的通信采用混合加密的方式进行传输，因此您不用担心你的账号密码泄漏的问题。

## 3. 应用授权权限说明
目前接入云塔账号通行证的应用可申请以下权限：
- 云塔平台：
  - 基础信息 [user:profile]
  - 邮箱 [user:email]
- 第三方平台
  - 编程猫账号UID [connect:codemao_uid]
  - PGAot账号UID [connect:pgaot_uid]
  - 神奇代码岛UID/DAO3账号UID [connect:dao3_uid]

云塔平台的【基础信息】权限可获取用户的以下信息：
- UID
- 昵称
- 用户名
- 头像
- 个人简介


## 4. 什么是第三方账号关系
第三方账号关系即您在云塔账号系统绑定的第三方账号的信息，不过由于云塔账号系统只保存的第三方账号的ID，因此应用也只能获取到您在第三方账号系统的ID

## 5. 神奇代码岛账号登录安全吗
神奇代码岛账号登录是绝对安全的，因为我们接入的是神奇代码岛官方的账号系统，我们**只能且只会**获取您在神奇代码岛的UID，昵称，头像，头像框信息
